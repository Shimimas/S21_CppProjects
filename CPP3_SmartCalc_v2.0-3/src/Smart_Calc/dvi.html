Запуск программы

Для запуска программы достаточно выполнить основную цель Makefile - make или make all.

Компиляция приложения внутри корневого каталога

Для компиляции приложения внутри корневого каталога достаточно выполнить цель build. Команда make build.

Создание исполняемого файла в каталоге "Desktop"

Для cоздания исполняемого файла в каталоге "Desktop" достаточно выполнить цель install. Команда make install.

Удаление исполняемого файла из каталогая "Desktop"

Для удаления исполняемого файла из каталогая "Desktop" достаточно выполнить цель uninstall. Команда make uninstall.
Также вместе с исполняемым файлом в корневом катологе будет удален каталог build.

Архивация проекта

Для архивации проекта достаточно выполнить цель dist. Команда make dist.

Модульное тестирование

Для модульногое тесирования достаточно выполнить цель test.Команда make test.

Стилевое тестирование

Для стилевого тесирования достаточно выполнить цель check.Команда make check.

Описание приложения

Программа поддерживает:
  • Вычисление выражений
  • Расчет кредитов
  • Расчет депозитов

После запуска приложения пользователю представлен графический интерфейс по взаимодействию с калькулятором.
В большей части окна приложения представлено поле для ввода выражений. В правой части функции
и кнопки для вычисления выражения, для активации разных режимов, для составления графика и другие.

Приятного пользования!
